Поместите сюда файл ffmpeg.exe, а в папку work файлы .mp3(если другой формат, можно изменить в 
splitAudioBySrt.py: "inFormat = "mp3") и .srt с одинаковым именем. При необходимости смените язык в dataset\lang.txt, по умолчанию: en. Далее start.bat. Для разделения субтитров по ролям используйте Aegisub.
