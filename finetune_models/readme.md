`Sub_to_dataset.py` нарежет звуковые файлы из папки `work` по субтитрам и конвертирует в `.wav`, также создаст файлы `metadata_eval.csv`(15% от датасета, минимум 1 запись) `metadata_train.csv`(85% от датасета) и поместит все в `dataset`, и еще создаст файл `Dialogs.txt`(про `Dialogs.txt` см. в основном `readme.md`)). При необходимости смените язык в `dataset\lang.txt`, по умолчанию: en. 

Для разделения субтитров по ролям(актерам, персонажам) используйте Aegisub. Для лучших результатов, выровняйте субтитры по звуковой волне в том же Aegisub или Subtitle Edit. Можно воспользоваться этим https://github.com/Le107/Voice_sorting/tree/main, а субтитры создать в Subtitle Edit.

`create_dataset.py` нужен если у вас уже имеется файл `Dialogs.txt`(обычно такой вытаскивают из игр на движке UE). Здесь остается только конвертировать звуковые файлы в '.wav' и поместистить в `dataset\wavs`, а файл `Dialogs.txt` рядом со скриптом. Скрипт по файлам из `dataset\wavs` сверит с `Dialogs.txt` и создаст файлы `metadata_train.csv` и `metadata_eval.csv` также 85/15%.
