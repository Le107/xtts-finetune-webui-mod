`Sub_to_dataset.py` нарежет звуковые файлы из папки `work` по субтитрам и конвертирует в `.wav`, также создаст файлы `metadata_eval.csv`(15% от датасета, минимум 1 запись) `metadata_train.csv`(85% от датасета) и поместит все в `dataset`, и еще создаст файл `Dialogs.txt`(про `Dialogs.txt` см. в основном `readme.md`)). При необходимости смените язык в `dataset\lang.txt`, по умолчанию: en. Для разделения субтитров по ролям используйте Aegisub. Для лучших результатов, выровняйте субтитры по звуковой волне в том же Aegisub или Subtitle Edit.
